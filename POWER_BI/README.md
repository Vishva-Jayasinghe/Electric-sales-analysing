# 📊 ElectroPulse Sales & Customer Analytics Dashboard

## 🧾 Project Overview
This project presents an end-to-end **Data Warehousing & Analytics solution** built using the **Medallion Architecture (Bronze → Silver → Gold layers)**. The objective is to transform raw data into actionable business insights through a well-structured data model and interactive dashboards.

---

## 🏗️ Data Architecture (Medallion Approach)

### 🥉 Bronze Layer
- Raw data ingestion from source systems
- Minimal transformations

### 🥈 Silver Layer
- Data cleaning and standardization
- Handling missing/null values
- Establishing relationships between datasets

### 🥇 Gold Layer
- Business-ready data model
- Aggregations and DAX measures
- Optimized for reporting

---

## 📐 Data Model
The solution follows a **Star Schema**:

- **Fact Table**
  - `sales`

- **Dimension Tables**
  - `customers`
  - `products`
  - `stores`
  - `calendar`

---

## 📊 Dashboard Features

### 🔹 Sales & Product Insights
- KPIs: Total Sales, Orders, Profit, Margin
- Monthly sales trend
- Sales by channel (Online vs Store)
- Top brands and product categories
- Delivery status tracking

### 🔹 Customer Insights
- Customer segmentation (Active, At Risk, Lost, Loyal)
- Top customers by sales
- Customer behavior trends
- Average Order Value & Repeat Rate
- Age group and regional analysis
